Start HereHow to use this portal

Work the query yourself first. Most tickets close without a single escalation.

  • Customer has a problem? Open the CX Playbook and search their words.
  • Need to check data? Grab a query from the SQL Tracking Library and run it in Metabase.
  • Chasing a gift or a sheet? The Sheets & IDs registry has the links and the tracking scheme.
  • Need to escalate? Post in #referrals-queries with both exporter IDs and screenshots — see the golden rules in the playbook.
